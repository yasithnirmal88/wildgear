# Wild Trail Gear - Management Portal

Wild Trail Gear is a premier outdoor gear rental service in Sri Lanka, based in Panadura. We provide high-quality equipment for camping, hiking, and exploring the untamed paths.

## Recent Updates

### Admin Panel Enhancements
*   **Mobile Responsiveness:** The admin panel is now fully responsive, featuring a toggleable off-canvas sidebar for mobile devices, a hamburger menu, and a responsive grid layout for dashboard statistics and inventory items.
*   **Table Usability:** Data tables (Rentals, Customers, Financials) are horizontally scrollable on smaller screens to prevent layout breakage.
*   **Brand Identity:** Integrated the official Wild Trail Gear logo into the admin login screen and the management dashboard sidebar.
*   **Isolated Layout:** The global website navigation, footer, and floating WhatsApp button are now hidden on admin routes to provide a focused, distraction-free management experience.
*   **Invoice Improvements:** The rental invoice modal has been optimized for mobile viewing, ensuring the printable bill displays correctly on all devices.
*   **Rental Management:** Added a missing "Remove" feature for items within the New Rental form for better usability.

## Tech Stack
*   **Framework:** Next.js
*   **Styling:** Tailwind CSS / Custom CSS
*   **Database:** Supabase (PostgreSQL)
